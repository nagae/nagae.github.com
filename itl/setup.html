2.3 BetterTouchToolをインストールしてみよう

BetteTouchToolは MagicTrackpad の様々なジェスチャに機能を割当てるツール．とても便利．これをインストールしてみよう．

2.3.1 通常の方法

  • http://blog.boastr.net/ から「Download BetterTouchTool」を選ぶ．
  • DLされた BetterTouchTool.zip をダブルクリックすると BetterTouchTool.app が作成される．
  • BetterTouchTool.app をアプリケーション・フォルダに移動する．

2.3.2 ターミナルを使う方法

$ cd ~/Downloads/                                   # ダウンロード・フォルダへ移動
$ curl -O http://www.boastr.de/BetterTouchTool.zip  # BetterTouchTool.zip ファイルを保存
$ unzip BetterTouchTool.zip                         # zip ファイルを解凍
$ mv BetterTouchTool.app/ /Applications/            # /Applications/ フォルダへ移動

2.3.3 注意事項

ー curl というのは Web ページにアクセスするためのコマンド．-O オプションはファイルを保存する時に使う．

  • .zip という拡張子は zip 形式で 圧縮 されたファイルであることを示している．
  • Mac のソフトウェアは「アプリケーション」というフォルダに格納されている．このフォルダは，コマンドライン上では /Applications/ に対応している．フォルダ名の前に / がついていることに注意しよう．
  • Mac のソフトウェアは，一見 .app と名前のついたファイルのように見えるが，実はフォルダになっている．例えば，BetterTouchTools の中は以下のようなフォルダ構成になっている． 
$ tree -L 2 BetterTouchTool.app/
BetterTouchTool.app/
└── Contents
    ├── Frameworks
    ├── Info.plist
    ├── MacOS
    ├── PkgInfo
    ├── Resources
    └── _CodeSignature

2.4 vi エディタの最低限の使い方を覚えよう

vi エディタは，ターミナル上で使える軽量かつ高機能なテキストエディタ．操作に癖があるが，ちょっとしたスクリプトや設定ファイルくらいであれば，後述する Emacs を立ち上げるよりも，これで書いた方が早い．

やりたいこと
rm コマンドはファイルを問答無用で削除してしまうので，「削除するたびに確認するオプション」 -i を常につけるようにしたい．
どうやるのか
ホームフォルダに .bash_profile を作り，そこに「 rm というコマンドは rm -i と読み替えなさい」という指示を書く．

2.4.1 vi を起動する

vi foo で foo というファイルを開く．ファイルが無ければ作る準備をしてくれる． 

$ vi ~/.bash_profile

2.4.2 読み替えの指示を書き込む

  • vi は起動時は閲覧モードになっているので，そのままでは編集できない．そこで，まずは i とタイプして挿入モードに切り替える．
  • 挿入モードになると右下に -- INSERT -- と表示される．そこで，以下を入力する．間違えたら backspace で戻ること． 
alias rm='rm -i'
  • 入力を終えたら保存する．まず Esc を押し，続いて「:w」とタイプして return する．これで保存される．
  • viを終了するには，Esc を押し，続いて「:q」とタイプする．

2.4.3 読み替えの指示を反映させる

$ vi ~/.bash_profile
